## Intro

### 설치

```
yarn install 또는 npm install
```

### 환경 변수 설정

```
 // .env.example 파일을 .env로 이름 변경합니다.

 REACT_APP_OPEN_WEATHER_API_KEY = { Open Weather apiKey }
```

### 실행

```
yarn dev 또는 npm run dev
```
